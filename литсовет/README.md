# Литсовет

ИИ-инструмент для написания прозы: многоагентный пайплайн (Архитектор → Прозаик ⇄ Оценщик → Стражи → Линейный редактор) с авторским контролем, памятью канона и экспортом в EPUB/DOCX/MD.

## Запуск

**Windows:** дважды кликните `deploy.bat` (проверит Node.js, запустит сервер, откроет браузер).

**Linux/macOS:**
```bash
chmod +x start.sh
./start.sh
```

**Вручную:**
```bash
node server.js
```

Откроется `http://localhost:8788` (порт можно переопределить переменной `PORT`).

Требуется Node.js 18+, https://nodejs.org — зависимостей и сборки нет.

## Данные

Проекты хранятся в браузере (IndexedDB) + синхронизируются на диск в `data/projects/` для доступа с других устройств. API-ключ никогда не сохраняется на диск — только в памяти браузера на время сессии.

## Перенос на другой компьютер

Запустите `build-deploy.bat` — соберёт архив на рабочем столе. На новом компьютере распакуйте и запустите `deploy.bat`.
